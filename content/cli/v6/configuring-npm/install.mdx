### 설명

공개 npm 레지스트리에서 패키지를 게시하고 설치하려면 Node 버전 관리자 또는 Node 설치 프로그램을 사용하여 Node.js와 npm 명령줄 인터페이스를 설치해야 합니다. **Node.js와 npm을 설치하기 위해 Node 버전 관리자를 사용하는 것을 강력히 권장합니다.** Node 설치 프로그램을 사용하는 것은 권장하지 않습니다. Node 설치 과정에서 npm을 로컬 권한이 있는 디렉토리에 설치하고, npm 패키지를 전역적으로 실행할 때 권한 오류를 일으킬 수 있기 때문입니다.

### npm과 Node.js 버전 확인하기

Node.js와 npm이 이미 설치되어 있는지 확인하고 설치된 버전을 확인하려면 다음 명령어를 실행하세요:

```
node -v
npm -v
```

### Node 버전 관리자를 사용하여 Node.js와 npm 설치하기

Node 버전 관리자를 사용하면 시스템에 여러 버전의 Node.js와 npm을 설치하고 전환할 수 있어, 다양한 npm 버전에서 애플리케이션을 테스트하여 다른 버전을 사용하는 사용자들에게도 작동하는지 확인할 수 있습니다.

#### OSX 또는 Linux Node 버전 관리자

- [nvm](https://github.com/creationix/nvm)
- [n](https://github.com/tj/n)

#### Windows Node 버전 관리자

- [nodist](https://github.com/marcelklehr/nodist)
- [nvm-windows](https://github.com/coreybutler/nvm-windows)

### Node 설치 프로그램을 사용하여 Node.js와 npm 설치하기

Node 버전 관리자를 사용할 수 없는 경우, Node 설치 프로그램을 사용하여 시스템에 Node.js와 npm을 모두 설치할 수 있습니다.

- [Node.js 설치 프로그램](https://nodejs.org/en/download/)
- [NodeSource 설치 프로그램](https://github.com/nodesource/distributions). Linux를 사용하는 경우 NodeSource 설치 프로그램을 사용하는 것이 좋습니다.

#### OS X 또는 Windows Node 설치 프로그램

OS X 또는 Windows를 사용하는 경우 [Node.js 다운로드 페이지](https://nodejs.org/en/download/)에서 설치 프로그램 중 하나를 사용하세요. **LTS**로 표시된 버전을 설치해야 합니다. 다른 버전은 아직 npm과 테스트되지 않았습니다.

#### Linux 또는 기타 운영 체제 Node 설치 프로그램

Linux 또는 다른 운영 체제를 사용하는 경우 다음 설치 프로그램 중 하나를 사용하세요:

- [NodeSource 설치 프로그램](https://github.com/nodesource/distributions) (권장)
- [Node.js 다운로드 페이지](https://nodejs.org/en/download/)의 설치 프로그램 중 하나

또는 많은 Linux 개발자가 선호하는 방식으로 Linux용 npm을 설치하려면 [이 페이지](https://nodejs.org/en/download/package-manager/)를 참조하세요.

#### 덜 일반적인 운영 체제

다양한 운영 체제에 Node.js를 설치하는 방법에 대한 자세한 정보는 [이 페이지][pkg-mgr]를 참조하세요.

[pkg-mgr]: https://nodejs.org/en/download/package-manager/
