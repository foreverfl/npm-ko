### 개요

```bash
npm whoami
```

참고: 이 명령은 작업 공간을 인식하지 않습니다.

### 설명

현재 로그인한 사용자의 npm 사용자 이름을 표시합니다.

토큰 기반 인증을 제공하는 레지스트리에 로그인한 경우, `/-/whoami` 레지스트리 엔드포인트에 연결하여 토큰과 연관된 사용자 이름을 찾아 표준 출력에 출력합니다.

기본 인증을 사용하는 레지스트리에 로그인한 경우, 인증 문자열의 `username` 부분만 단순히 출력합니다.

### 설정

#### `registry`

- 기본값: "https://registry.npmjs.org/"
- 유형: URL

npm 레지스트리의 기본 URL입니다.
