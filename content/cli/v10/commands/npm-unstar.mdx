### 개요

```bash
npm unstar [<package-spec>...]
```

참고: 이 명령은 작업 공간을 인식하지 않습니다.

### 설명

패키지를 "unstar"하는 것은 [`npm star`](/cli/v10/commands/npm-star)의 반대로, 즐겨찾는 패키지 목록에서 항목을 제거합니다.

### 추가 정보

즐겨찾는 패키지를 관리하는 데 도움이 되는 추가 명령어들이 있습니다:

#### 스타

[`npm star`](/cli/v10/commands/npm-star)를 사용하여 패키지에 "스타"를 줄 수 있습니다.

#### 스타 목록 보기

[`npm stars`](/cli/v10/commands/npm-stars)를 사용하여 스타를 준 모든 패키지를 볼 수 있습니다.

### 설정

#### `registry`

- 기본값: "https://registry.npmjs.org/"
- 유형: URL

npm 레지스트리의 기본 URL입니다.

#### `unicode`

- 기본값: Windows에서는 false, Mac/Unix 시스템에서는 `LC_ALL`, `LC_CTYPE`, 또는 `LANG` 환경 변수에 의해 정의된 유니코드 로케일이 있는 경우 true
- 유형: Boolean

true로 설정되면 npm은 트리 출력에 유니코드 문자를 사용합니다. false일 때는 유니코드 글리프 대신 아스키 문자를 사용합니다.

#### `otp`

- 기본값: null
- 유형: null 또는 String

이것은 2단계 인증기에서 제공하는 일회용 비밀번호입니다. 패키지를 게시하거나 `npm access`로 패키지 권한을 변경할 때 필요합니다.

설정되지 않았고 레지스트리 응답이 일회용 비밀번호를 요구하는 경우, npm은 명령줄에서 입력을 요청합니다.
