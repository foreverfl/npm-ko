## 보안 키 흐름을 사용하여 명령줄에서 로그인하기

1. 명령줄에서 [`npm login`][login] 명령을 입력합니다.

2. 메시지가 표시되면 사용자 이름, 비밀번호 및 이메일 주소를 제공합니다.

   ```
   user@host:~$ npm login
   npm notice Log in on https://registry.npmjs.org/
   Username: mona
   Password:
   Email: (this IS public) mona@github.com
   npm notice Open https://www.npmjs.com/login/913c3ab1-89a0-44bd-be8d-d946e2e906f0 to use your security key for authentication or enter OTP from your authenticator app
   ```

3. 보안 키를 구성한 경우 명령줄에 표시된 URL을 엽니다. 또는 모바일 인증기를 구성한 경우 6단계로 건너뜁니다.

4. *보안 키 사용*을 클릭하고 브라우저별 단계에 따라 인증합니다.

   <Screenshot src="/getting-started/setting-up-your-npm-user-account/2fa-use-security-key.png" alt="보안 키 프롬프트를 보여주는 스크린샷" />

5. 생성된 토큰을 복사합니다.

   <Screenshot src="/getting-started/setting-up-your-npm-user-account/webauthn-cli-login-token.png" alt="WebAuthn을 사용한 CLI 로그인 시 생성된 샘플 토큰을 보여주는 스크린샷" />

6. CLI 프롬프트에 일회용 비밀번호를 입력합니다.

   <Prompt>Enter one-time password:</Prompt>

## `--auth-type=web`을 사용하여 명령줄에서 로그인하기

npm 8.14.0 이상은 브라우저를 통한 로그인 흐름을 지원합니다. 이는 npm 9에서 npm 공개 레지스트리의 기본 동작이 될 것입니다.

### 기존 브라우저 세션이 있는 경우

1. 명령줄에서 [`npm login --auth-type=web`][login] 명령을 입력합니다.

2. 메시지가 표시되면 "ENTER"를 눌러 브라우저를 열고 로그인 흐름을 시작하거나 명령줄에 표시된 URL을 클릭합니다.

   ```
   user@host:~$ npm login
   npm notice Log in on https://registry.npmjs.org/
   Authenticate your account at:
   https://www.npmjs.com/login?next=/login/cli/b1a2f96a-ce09-4463-954c-c99f6773b922
   Press ENTER to open in the browser...
   ```

3. *보안 키 사용*을 클릭하고 브라우저별 단계에 따라 인증합니다.

   <Screenshot src="/getting-started/setting-up-your-npm-user-account/2fa-use-security-key.png" alt="보안 키 프롬프트를 보여주는 스크린샷" />

   _참고: TOTP 사용을 구성한 경우 대신 TOTP 프롬프트가 표시됩니다_

### 기존 브라우저 세션이 없는 경우

1. 명령줄에서 [`npm login --auth-type=web`][login] 명령을 입력합니다.

2. 메시지가 표시되면 "ENTER"를 눌러 브라우저를 열고 로그인 흐름을 시작하거나 명령줄에 표시된 URL을 클릭합니다.

   ```
   user@host:~$ npm login
   npm notice Log in on https://registry.npmjs.org/
   Authenticate your account at:
   https://www.npmjs.com/login?next=/login/cli/b1a2f96a-ce09-4463-954c-c99f6773b922
   Press ENTER to open in the browser...
   ```
