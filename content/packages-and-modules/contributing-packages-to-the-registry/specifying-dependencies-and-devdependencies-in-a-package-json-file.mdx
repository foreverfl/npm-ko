---
title: package.json 파일에서 dependencies와 devDependencies 지정하기
---

프로젝트가 의존하는 패키지를 지정하려면 패키지의 [`package.json`][pkg-json] 파일에 `"dependencies"` 또는 `"devDependencies"`로 나열해야 합니다. 여러분(또는 다른 사용자)이 `npm install`을 실행하면, npm은 `package.json`에 나열된 dependencies와 devDependencies 중 나열된 [시맨틱 버전][semver] 요구사항을 충족하는 것을 다운로드합니다. 어떤 버전의 패키지가 설치될지 확인하려면 [semver 계산기][semver-calc]를 사용하세요.

- `"dependencies"`: 프로덕션 환경에서 애플리케이션에 필요한 패키지.
- `"devDependencies"`: 로컬 개발 및 테스트에만 필요한 패키지.

## `package.json` 파일에 dependencies 추가하기

명령줄에서 또는 `package.json` 파일을 직접 편집하여 dependencies를 추가할 수 있습니다.

### 명령줄에서 `package.json` 파일에 dependencies 추가하기

명령줄에서 `package.json` 파일에 dependencies와 devDependencies를 추가하려면 패키지의 루트 디렉토리에서 dependencies의 경우 `--save-prod` 플래그(또는 `-S`, `npm install`의 기본 동작)를 사용하여, devDependencies의 경우 `--save-dev` 플래그(또는 `-D`)를 사용하여 설치할 수 있습니다.

`package.json` 파일의 `"dependencies"` 속성에 항목을 추가하려면 명령줄에서 다음 명령을 실행하세요:

```
npm install <package-name> [--save-prod]
```

`package.json` 파일의 `"devDependencies"` 속성에 항목을 추가하려면 명령줄에서 다음 명령을 실행하세요:

```
npm install <package-name> --save-dev
```

### `package.json` 파일 수동 편집하기

`package.json` 파일에 dependencies를 추가하려면 텍스트 편집기에서 각 dependency의 이름과 [시맨틱 버전][semver]을 참조하는 `"dependencies"`라는 속성을 추가하세요:

```
{
  "name": "my_package",
  "version": "1.0.0",
  "dependencies": {
    "my_dep": "^1.0.0",
    "another_dep": "~2.2.0"
  }
}
```

`package.json` 파일에 devDependencies를 추가하려면 텍스트 편집기에서 각 devDependency의 이름과 [시맨틱 버전][semver]을 참조하는 `"devDependencies"`라는 속성을 추가하세요:

```
"name": "my_package",
"version": "1.0.0",
"dependencies": {
  "my_dep": "^1.0.0",
  "another_dep": "~2.2.0"
},
"devDependencies" : {
  "my_test_framework": "^3.1.0",
  "another_dev_dep": "1.0.0 - 1.2.0"
}
```
