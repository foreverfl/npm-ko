---
title: 패키지 이름 지침
---

패키지의 이름을 선택할 때는 다음과 같은 이름을 선택하세요:

- 고유한 이름
- 설명적인 이름
- [npm 정책 지침][policies]을 충족하는 이름. 예를 들어, 패키지에 공격적인 이름을 붙이지 말고, 다른 사람의 상표명을 사용하거나 [npm 상표 정책][npm-trademark]을 위반하지 마세요.

또한, [**스코프가 없는** 패키지][create-unscoped]의 이름을 선택할 때는 다음과 같은 이름을 선택하세요:

- 다른 사람이 이미 소유하고 있지 않은 이름
- 다른 패키지 이름과 유사하게 철자가 쓰이지 않은 이름
- 저작권에 대해 다른 사람들을 혼란스럽게 하지 않을 이름

[policies]: https://www.npmjs.com/policies
[npm-trademark]: https://docs.npmjs.com/policies/disputes#trademarks
[create-unscoped]: creating-and-publishing-unscoped-public-packages
