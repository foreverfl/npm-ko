---
title: 패키지 README 파일에 대하여
---

다른 사람들이 npm에서 여러분의 패키지를 쉽게 찾고 자신의 프로젝트에서 여러분의 코드를 사용하는 데 좋은 경험을 할 수 있도록, 패키지 디렉토리에 README 파일을 포함하는 것을 권장합니다. README 파일에는 패키지의 코드를 설치, 구성 및 사용하는 방법에 대한 지침과 사용자가 유용하다고 생각할 수 있는 기타 정보를 포함할 수 있습니다. README 파일은 패키지 페이지에 표시됩니다.

npm 패키지 README 파일은 패키지의 루트 레벨 디렉토리에 있어야 합니다.

## README.md 파일 생성 및 패키지에 추가하기

1. 텍스트 편집기에서 패키지 루트 디렉토리에 `README.md`라는 파일을 만듭니다.
2. `README.md` 파일에 패키지에 대한 유용한 정보를 추가합니다.
3. `README.md` 파일을 저장합니다.

<Note>

**참고:** 파일 확장자 `.md`는 Markdown 파일을 나타냅니다. Markdown에 대한 자세한 정보는 GitHub 가이드 "[Mastering Markdown](https://guides.github.com/features/mastering-markdown/#what)"을 참조하세요.

</Note>

## 기존 패키지 README 파일 업데이트하기

README 파일은 패키지의 새 버전을 게시할 때만 패키지 페이지에서 업데이트됩니다. README 파일을 업데이트하려면:

1. 텍스트 편집기에서 `README.md` 파일의 내용을 업데이트합니다.

2. `README.md` 파일을 저장합니다.

3. 명령줄에서 패키지 루트 디렉토리에서 다음 명령을 실행합니다:

   ```
   npm version patch
   npm publish
   ```
