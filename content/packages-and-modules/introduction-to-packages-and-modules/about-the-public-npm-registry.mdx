---
title: 공개 npm 레지스트리에 대하여
---

공개 npm 레지스트리는 JavaScript 패키지의 데이터베이스로, 각 패키지는 소프트웨어와 메타데이터로 구성되어 있습니다. 오픈 소스 개발자들과 기업의 개발자들은 npm 레지스트리를 사용하여 전체 커뮤니티나 조직의 구성원들에게 패키지를 기여하고, 자신의 프로젝트에서 사용할 패키지를 다운로드합니다.

레지스트리를 시작하려면, [npm 계정을 생성][signup]하고 "[시작하기][getting-started]"와 [CLI][cli] 문서를 확인하세요.

[cli]: /cli-documentation
